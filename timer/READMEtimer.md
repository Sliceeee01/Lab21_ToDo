## Описание
Создано приложение "Timer" с использованием HTML, CSS и JavaScript.

### Функционал:
- Отображение времени в формате ЧЧ:ММ:СС
- Кнопка "Старт" - запуск отсчёта
- Кнопка "Стоп" - остановка отсчёта
- Кнопка "Сброс" - обнуление времени
